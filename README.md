# Game-Guide-Hub
Огляд проекту

GameGuideHub — веб-платформа для швидкого пошуку актуальних гайдів по відеоіграх. Система дозволяє гравцям знаходити корисну інформацію (білди, стратегії, поради) без необхідності переглядати довгі відео або форуми.

Проєкт спрямований на об’єднання гайдів в одному місці з фільтрацією за грою, роллю, рівнем складності та актуальністю.

🩹 Проблема
Гравці витрачають багато часу на пошук інформації:
гайди розкидані по YouTube, форумах і сайтах
інформація часто застаріла після оновлень
складно знайти короткий і зрозумілий гайд
Автори гайдів не мають зручної платформи для структурованої подачі інформації.

🔧 Pitch
Система дозволяє:
Гравцям:
знаходити гайд за 1–2 хвилини
фільтрувати по грі, персонажу, ролі
бачити тільки актуальні гайди
Авторам:
створювати структуровані гайди
отримувати оцінки та відгуки
оновлювати контент після патчів
Адміністрації:
модерувати контент
відслідковувати популярність гайдів
Цільова аудиторія: геймери (новачки і середній рівень), автори контенту.

♻️ MVP (Мінімально життєздатний продукт)
Ключовий сценарій:
Автор створює гайд
Гравець заходить на сайт
Обирає гру / персонажа
Переглядає гайд

👷 Стейкхолдери
Гравець
Очікування: швидко знайти зрозумілий гайд
Критерій успіху: знаходить потрібну інформацію за кілька кліків

Автор гайдів
Очікування: зручне створення контенту
Критерій успіху: гайди переглядають і оцінюють
Адміністрація
Очікування: якісний контент
Критерій успіху: мінімум скарг і спаму

📈 Мета першої ітерації
Запустити базову версію:
автор створює гайд → користувач знаходить → читає

🔨 Критерії успіху
Гравець знаходить гайд ≤ 1 хвилини
≥ 70% користувачів переглядають гайд до кінця
≥ 10 гайдів створено в тестовий період
🔀 Детальний користувацький сценарій
Умови: користувач шукає гайд

🚸 Кроки
Вхід на сайт
Вибір гри
Пошук персонажа / теми
Вибір гайду
Перегляд
Результат: користувач отримує потрібну інформацію
🔥 Гіпотези
📝 Гіпотеза 1: Користувачі хочуть короткі гайди
Цінність: економія часу
Метрики:
середній час пошуку ≤ 1 хв
≥ 70% дочитують гайд
📝 Гіпотеза 2: Актуальність важливіша за кількість
Цінність: тільки корисна інформація
Метрики:
≥ 50% гайдів оновлюються після патчів
🛂 Невизначеності
1. Чи будуть люди писати гайди?
Рішення:
проста форма створення
лайки / рейтинг
2. Як перевіряти актуальність?
Рішення:
позначка “оновлено під патч”
дата останнього редагування
3. Чи потрібні відео?
Рішення:
тільки текст + картинки в MVP

👔 Definition of Success
Ітерація успішна якщо:
Кількісні:
≥ 70% користувачів знаходять гайд
≥ 10 активних гайдів
Якісні:
≥ 80% відгуків позитивні
Технічні:
сайт працює без збоїв 7 днів
